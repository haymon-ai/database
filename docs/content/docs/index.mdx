---

Database MCP is a Model Context Protocol (MCP) server built in Rust with zero runtime dependencies. It provides AI assistants with safe, read-only access to your databases.

## Supported Databases

- **MySQL**
- **MariaDB**
- **PostgreSQL**
- **SQLite**

## Features

- Single binary with no runtime dependencies
- Read-only mode by default for safety
- Stdio and HTTP transport modes
- Works with Claude Desktop, Cursor, and other MCP clients

Check out the [GitHub repository](https://github.com/haymon-ai/database-mcp) for installation instructions and usage examples.
